# Allison's Automatic Transmission Parts - Website

This is a complete, responsive 5-page static website for Allison's Automatic Transmission Parts, built with HTML5, CSS3, and Vanilla JavaScript.

## 🚀 Features
- **Responsive Design**: Works perfectly on Desktop, Tablet, and Mobile.
- **AOS Animations**: Smooth scroll animations.
- **Swiper.js Carousel**: For Google Reviews.
- **Formspree Integration**: Contact form is ready to be linked to an email.
- **Google Maps**: Embedded location for the Houston shop.
- **Filterable Catalog**: Vanilla JS component filtering for parts.

## 📂 File Structure
- `index.html`: Homepage with hero, features, featured parts, and review snippets.
- `parts.html`: Filterable catalog of auto parts.
- `about.html`: Story of the shop and the staff.
- `reviews.html`: Detailed Google reviews page.
- `contact.html`: Contact info, form, and Google Maps embed.
- `css/style.css`: All styling (Mobile-First approach).
- `js/script.js`: Interactions (Preloader, Menu, Filtering, Back-to-Top, Swiper).

## 🛠️ Deployment Instructions
This website is ready to be deployed on any static hosting provider like Netlify, Vercel, or GitHub Pages.

**To deploy to Netlify:**
1. Create a Netlify account.
2. Drag and drop the `allisons-transmission` folder into the Netlify dashboard.
3. Your site will be live instantly!

## ⚙️ Configuration (Contact Form)
To receive emails from the contact form:
1. Go to Formspree (https://formspree.io/)
2. Create a form and get your endpoint URL.
3. Open `contact.html`.
4. Replace `action="https://formspree.io/f/YOUR_FORM_ID"` with your actual Formspree endpoint URL.
